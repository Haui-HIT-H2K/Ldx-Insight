# Thuật ngữ & viết tắt

| Thuật ngữ | Mô tả |
|-----------|-------|
| **Ldx-Insight** | Nền tảng tích hợp và phân tích dữ liệu mở do nhóm phát triển. |
| **OpenHub Crawler** | Bộ công cụ Python thu thập, chuẩn hóa dữ liệu từ các cổng open data. |
| **Dataset** | Tập dữ liệu được chuẩn hóa, lưu trong MongoDB và cung cấp qua API. |
| **Resource** | File cụ thể thuộc một dataset (CSV, XLSX, JSON, API endpoint). |
| **Connector** | Thành phần trong crawler kết nối tới một portal cụ thể (dong-thap, hcm). |
| **Portal** | Cổng dữ liệu mở của tỉnh/thành hoặc tổ chức. |
| **MongoDB Atlas** | Dịch vụ MongoDB Database-as-a-Service dùng cho dự án. |
| **EC2** | Elastic Compute Cloud (AWS) – nơi deploy backend Spring Boot. |
| **Vercel** | Nền tảng deploy frontend Nuxt và doc site. |
| **Cloudflare** | Dịch vụ CDN/DNS/Firewall bảo vệ và tăng tốc website. |
| **CI/CD** | Continuous Integration / Continuous Deployment – quy trình tự động build & deploy. |
| **JWT** | JSON Web Token – cơ chế xác thực người dùng (backend). |
| **Swagger / OpenAPI** | Bộ tiêu chuẩn mô tả REST API, tích hợp qua Springdoc. |
| **Pinia** | State management cho Vue/Nuxt. |
| **Nuxt UI** | Bộ component UI cho Nuxt 4. |
| **Tailwind CSS** | CSS utility-first framework dùng để styling. |
| **APScheduler** | Thư viện Python lập lịch task cho crawler. |
| **Tenacity** | Thư viện retry logic cho HTTP request (crawler). |
| **Normalized Data** | Dữ liệu đã được chuyển đổi sang định dạng chuẩn (JSON) để phân tích. |
| **Download Count** | Chỉ số lượt tải dataset, lưu trong MongoDB. |
| **Roadmap** | Kế hoạch phát triển dài hạn của dự án. |
| **DevOps** | Thực tiễn kết hợp Development và Operations (CI/CD, monitoring, IaC). |
