# Tổng quan Backend

## Mục tiêu

Tài liệu backend được chia thành hai phần chính:

- **Backend Spring Boot** – dịch vụ RESTful phục vụ ứng dụng và API.
- **Backend Python (OpenHub Crawler)** – hệ thống thu thập dữ liệu từ các cổng dữ liệu mở.

## Danh mục

- [Backend Spring Boot](backend-spring-boot/overview)
- [Backend Python](backend-python/overview)

## Gợi ý đọc

1. Bắt đầu với phần **Spring Boot** để hiểu API và bảo mật.
2. Tiếp tục với phần **Python Crawler** để biết quy trình thu thập dữ liệu nguồn.
3. Tùy theo nhu cầu, tham khảo các chương về kiến trúc, công nghệ và hướng dẫn cài đặt của từng phần.
