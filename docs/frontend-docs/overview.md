# Tổng quan tài liệu Frontend

Tài liệu frontend được chia thành các chương mô tả kiến trúc Nuxt, công nghệ và hướng dẫn vận hành ứng dụng web của **Ldx-Insight**.

## Danh mục chính

- [Frontend Nuxt — Tổng quan](frontend-nuxt/overview)
- [Frontend Nuxt — Thiết kế hệ thống](frontend-nuxt/architecture)
- [Frontend Nuxt — Công nghệ sử dụng](frontend-nuxt/technologies)
- [Frontend Nuxt — Hướng dẫn cài đặt](frontend-nuxt/installation)

## Gợi ý đọc

1. Bắt đầu với phần **Tổng quan** để hiểu mục tiêu và phạm vi.
2. Xem tiếp **Thiết kế hệ thống** để nắm được cấu trúc thư mục, luồng dữ liệu.
3. Tham khảo **Công nghệ sử dụng** trước khi tùy chỉnh hoặc nâng cấp.
4. Làm theo **Hướng dẫn cài đặt** để chạy thử hoặc triển khai.

Các tài liệu chi tiết về từng thư viện (Nuxt UI, Pinia, Tailwind, v.v.) vẫn được giữ trong mục `Công nghệ sử dụng` để tra cứu nhanh.
